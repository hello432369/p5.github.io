    2.设置

    程序运行前,可以设置很多事情,比如全屏模式等等

    createCanvas(windowWidth, windowHeight);    //全屏模式

    3.存储

    数据是临时存储在内存当中,创建变量意味着在内存中申请一个空间用于存放我们的数据,一般是通过变量参与计算.内存中的数据随着软件的关闭而丢失,想要保存数据,就需要保存为文件,文件本质上是一个特殊的数据库.

    let myArray = [ ];              //创建一组变量,可存放多个变量

    4.输入

    把数据直接写在程序,这样每次运行都是固定的结果.有时只想修改一些参数,不想给别人看代码,可以提供用户界面(即UI),便于他人使用.不同的数据类型用用不同的UI输入方式

    createColorPicker();  //调整颜色

    5.处理

    算法是程序的核心,现实中我们叫它"黑匣子",算法的本质是思考方式,在有限的时间复杂度(高效率)和空间复杂度(低成本)内达成目标.

    运算 

    数字：+ * - /     //数学运算
    控制 
    while( ){ }    //如果(满足小括号内的条件),{执行无限次大括号中的内容}

    功能 

    变量名
变量名.value()   //使用键盘输入的数据
变量名.color()   //使用色彩盘中的数据

int()           //将其他数据类型转为整数类型,转后可以参与计算

    6.输出

    经过存储,输入,处理之后, 输出结果的方法多样

    print(内容);               //按F12观察计算结果

    7.案例

    知识点是理论,还要通过实践去检验理论,并生成自己的艺术作品

    let mydata;
let mycolor; //创建存储空间

function setup() {
    createCanvas(369, 369);

    mydata = createInput("6");
    mycolor = createColorPicker();   //获取颜色数据,并存储道内存空间
}

function draw() {

    fill( mycolor.color() );  //处理颜色数据,并输出颜色数据

    if (mouseIsPressed) {
        text(mydata.value() , mouseX , mouseY);
    }
}
