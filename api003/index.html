    存储

    序言

    数据是临时存储在内存当中,创建变量意味着在内存中申请一个空间,用于存放我们的数据,一般是通过变量去参与计算.

    内存中的数据随着软件的关闭而丢失,想要长久保存数据,就需要保存为文件,文件本质上是一个特殊的数据库.

    单一存储

    let     //可创建所有数据类型:数字,字符,布尔,数组,向量,对象,类
const   //常数类型
class   //类(存放相同类型的数据集合)
object  //对象

    多个存储

    let myArray = [1, 2, 3, 4];              //一维数组
let myArray = [[1, 2], [3, 4]];           //二维数组
let myArray = [[[1, 2], [3, 4]], [[5, 6], [7, 8]]];;        //三维数组
let myList = createList();      //列表
let myDict = createDict();      //字典
let myvector = createVector();      //向量

    类型转换

    float()     //转小数
int()       //转整数
str()       //转字符串
boolean()   //转布尔
byte()      //转比特
char()      //转字符
unchar()    //字符转数字
hex()       //转十六进制
unhex()     //十六进制转数字
